# 课程论文任务规划

## 1. 当前任务边界

当前只撰写《文献检索与论文写作》课程论文，不制作 PPT。后续所有规划、检索、写作、排版和检查均围绕课程论文成稿展开。

课程论文必须满足任务书要求：

- 面向 23 级医学检验技术专业。
- 自主命题，且不得与当前选题表中已有题目重复。
- 正文字数不少于 5000 字。
- 论文应体现文献检索策略、文献管理与阅读分析、学术论文结构与写作规范、学术伦理和综合反思。
- 高分文献综述目标为 15 篇以上权威文献，建议最终引用 20-25 篇。

### 2.1 不再采用的题目

原因：选题表中已有“肾功能指标在慢性肾病分期中的应用探讨”，另有“血清肌酐与尿素氮检测在肾功能评价中的应用研究”和“尿微量白蛋白在早期肾损伤中的临床应用研究”。原拟题目虽然表述更细，但研究方向与已有题目过近，存在被认定为重复或高度相似的风险。

| 维度 | 理由 |

| 医学检验专业关联 | sFlt-1、PlGF 及其比值属于血清生物标志物检测，涉及检验方法、报告解释、质量控制和临床决策支持 |

## 4. 研究问题

主研究问题：

在疑似或高风险子痫前期孕妇中，sFlt-1/PlGF 比值在早期预测、短期排除、辅助诊断和临床风险分层中具有怎样的检验学价值？

分问题：

1. sFlt-1、PlGF 与子痫前期发病机制之间有什么联系？
2. sFlt-1/PlGF 比值相较单独检测 sFlt-1 或 PlGF 有哪些优势和局限？
3. 现有指南、临床研究和系统综述对该比值的预测价值有哪些共识？
4. 不同孕周、检测平台、截断值、样本处理和人群差异会如何影响结果解释？
5. 医学检验人员在报告、质控和临床沟通中应注意哪些问题？

### 6.5 纳入和排除标准

纳入标准：

- 与 sFlt-1、PlGF 或 sFlt-1/PlGF 比值检测直接相关。
- 研究对象为疑似、高风险或已诊断子痫前期孕妇。
- 文献类型为指南、诊断指导、系统综述、Meta 分析、前瞻性队列研究、多中心研究、临床应用研究或高质量综述。
- 优先 2016-2026 年文献，保留必要经典研究。

排除标准：

- 与子痫前期预测或诊断无直接关系。
- 只讨论基础机制但没有临床检验应用价值。
- 非学术网页、商业宣传、重复发表或证据质量较低的病例报告。
- 无法获取基本文献信息或引用信息不完整。

## 7. 文献管理与证据矩阵

目标：

- 初筛文献不少于 40 篇。
- 精读核心文献 20-25 篇。
- 最终参考文献不少于 15 篇，建议 20 篇以上。

核心文献分类：

| 类型 | 数量目标 | 用途 |
| --- | ---: | --- |
| 指南/诊断指导 | 3-5 篇 | 支撑临床应用建议和检测定位 |
| 系统综述/Meta 分析 | 3-5 篇 | 概括诊断或预测价值 |
| 前瞻性或多中心研究 | 5-8 篇 | 支撑短期预测、排除和截断值讨论 |
| 中文临床检验文献 | 4-6 篇 | 体现国内应用情况和医检专业视角 |
| 方法学或质控文献 | 2-4 篇 | 支撑检测平台、样本、质量控制和报告解释 |

每篇核心文献记录字段：

- 作者与年份
- 文献类型
- 研究对象
- 检测指标
- 主要结论
- 局限性
- 拟用于论文位置
- DOI、PMID 或数据库链接

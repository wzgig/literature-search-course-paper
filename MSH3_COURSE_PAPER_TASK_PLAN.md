# MSH3基因检测课程论文任务规划

## 任务边界

- 课程：《基因检测与解读》。
- 论文题目：`MSH3基因的检测与分析`。
- 基因编号：任务书表格中第 81 号，基因为 `MSH3`。
- 本轮交付：课程论文初稿、正式 Word 排版稿、检索记录、证据矩阵、精读笔记和来源记录。
- 不在本轮制作 PPT。任务书虽列出 PPT 材料，但用户当前要求仅完成课程论文。

| 要求 | 对应执行方案 |
| --- | --- |
| 3000 字以上 | 正文按 7000 字以上撰写，明显高于下限 |
| 资料搜集全面 | 至少纳入 10 篇期刊论文，其中英文文献超过 5 篇 |

| 格式规范 | 使用课程论文 Word 模板结构，生成封面、目录、摘要、正文、图表、参考文献和查重报告附页 |

## 核心研究问题

1. MSH3 在 DNA 错配修复系统中的功能是什么，为什么其缺陷会与肿瘤易感和微卫星改变相关？
2. MSH3 检测在遗传性结直肠腺瘤性息肉病、早发结直肠癌和肿瘤错配修复评价中分别有什么价值？
3. 对 MSH3 进行检测时，如何根据临床目的选择 PCR/Sanger、NGS panel、WES/WGS、CNV 分析、MSI/EMAST 和 IHC 等方法？
4. MSH3 检测结果应如何解释，如何避免把单个 VUS 或单项肿瘤检测结果过度解读为明确诊断？
5. 基因检测报告进入临床和家系沟通后，如何处理隐私、歧视、知情同意和人类遗传资源合规问题？

## 纳入与排除标准

纳入：

- MSH3 基因功能、错配修复、微卫星不稳定性、EMAST 或 MutSβ 相关研究；
- MSH3 相关腺瘤性息肉病、早发结直肠癌、遗传性肿瘤易感研究；
- NGS panel、WES/WGS、变异解释和 ClinVar/ACMG 相关资料；
- 可提供 DOI、PMID、官方链接或明确出处的来源；
- 近 5 年文献优先，经典发现性文献作为必要基础保留。

排除：

- 仅在全文参考文献中偶然出现 MSH3、正文不讨论 MSH3 的文献；
- 搜索结果摘要与主题无关的重复扩增疾病文献，除非用于说明 MSH3 在 DNA 修复中的前沿机制；
- 无法核验题名、作者、年份、DOI/PMID 或官方链接的资料；
- 与论文主题无关的商业检测宣传材料。
